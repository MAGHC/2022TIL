### 정렬 (sort)

배열을 정열 

let array = [2,6,5]

array.sort()

console.log(array)

[2,5,6]

하지만 이것은 숫자 정렬이 아님  문자로 생각하고 정렬 

array.sort(function(a,b){
    return a-b;
})


이게 숫자 정렬 

why? 

a,b = array 안에있던 하나하나의 데이터 

2, 5 

2-5 = -2  음수면 

a를 왼쪽으로 보내는 식으로 동작



2,5 해봤으니까 다음엔 5,6 (a,b)

-1 

5를 왼쪽으로 


2,6 이랑도 해봐서 -나올때마다 a 부분을 왼쪽으로 안나오면 오른쪽으로 해서 정렬 


보통 외워서 쓴다고 함 


b-a 하면 자연히 거꾸로 내림차순

a-b 오름차순
