 ### npm & yarn 

 npm : node.js 를 설치하면 사용 가능 각종 라이브러리 설치 가능 


 npm install -g @vue/cli

 npm이 잘 안될시! yarn 이라는 대체품이 있다 
 google에서 yarn 다운로드 
 터미널에서 
 yarn global add @vue/cli 

 yarn을 쓰는이유 yarn이 좀 더 안정적이고 빠름 & 에러가 덜함
