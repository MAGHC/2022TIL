### performance

성능 개선 확인

녹화로 하고싶은 액션 취해서 성능 확인 가능

layout shift 와 같이 알아서 어떤 요소에서 어떤 성능저해 요소가 나타났는지 알려줌

사용자의 완만한 경험을 위해선 1초당 60프레임이 나와야됨
한프레임이 16ms 동안 끝내야되는데 그보다 길다면 성능에 관해서 고려해볼것
