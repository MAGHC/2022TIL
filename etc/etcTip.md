### 멀터 터미널로 커밋 남기기

최근 유데미를 보다가 터미널 두개를 동시에 쓰는 걸 봤다.

물론 npm install 같은 건 끄고 해야 제대로 추가가 되고 껏다 켜야되겠지만

터미널 두개를 켜놓고 하나는 npm start 해놓고 다른 하나에서는 git add .

commit 하는 방식으로 작업이 가능했다

이게 될까? 했는데 되서 신기
