### context api

리덕스랑 비슷한 거

let context1 = createContext()

이렇게 생성

원하는 컴포넌트 해당 context 컴포넌트로 감싸야됨

전체를 관장하는 파일에서 ㅇ 라우트 설정 다되어있는곳에서 하면됨

당연히 임폴트 해야되고
훅처럼

useContext(context1)

context1 의 내용에 따라 구조분해할당 가능

### 특징

쓸데 없는 거 까지 재랜더링

context 생성을 재사용해야되는 컴포넌트에 선언했다면 해당 컴포넌트 재사용이 어려움
